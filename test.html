🚀 Sidebar 功能测试页面

测试说明：

1. 点击浏览器扩展图标（在工具栏中）来打开/关闭 sidebar

2. 使用快捷键 Ctrl + M (Windows/Linux) 或 Cmd + M (Mac) 来切换 sidebar

3. 按 ESC 键关闭 sidebar

4. 点击遮罩层（sidebar 外部区域）关闭 sidebar

关于这个功能

这个 sidebar 实现了：

  • ✅ 从右侧滑入的动画效果
  • ✅ 支持所有网页（<all_urls>）
  • ✅ 使用 React + shadcn/ui 组件构建
  • ✅ 响应式设计，适配不同屏幕尺寸
  • ✅ 多种关闭方式（ESC 键、点击遮罩、点击关闭按钮）
  • ✅ 跨浏览器兼容（基于 WXT 框架）
  • ✅ 提示词管理功能界面

技术实现

该 sidebar 使用了以下技术栈：

  • WXT 框架：现代化的浏览器扩展开发框架
  • React 19：用于构建用户界面
  • shadcn/ui：高质量的 UI 组件库
  • Tailwind CSS：实用优先的 CSS 框架
  • Lucide Icons：美观的图标集

样式参考

界面设计参考了 OpenPrompt 的设计风格，包括：

  • 现代化的卡片布局
  • 清晰的导航结构
  • 一致的颜色方案
  • 流畅的动画过渡

💡 提示：

如果看不到扩展图标，请确保已经加载了扩展，并且检查浏览器的扩展管理页面。
